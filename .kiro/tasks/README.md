# Task Registry

## Overview
Direktori ini berisi definisi task individual menggunakan format TASK-XXXX.
Setiap task adalah unit kerja independen yang dapat dieksekusi tanpa membaca seluruh repository.

## Usage

1. Buat file baru: `TASK-XXXX.md` (gunakan template dari `.kiro/templates/task.md`)
2. Isi semua field termasuk `Files to Read` dan `Files to Modify`
3. Deklarasikan dependencies secara eksplisit
4. Setelah selesai, update status ke `[COMPLETED]`
